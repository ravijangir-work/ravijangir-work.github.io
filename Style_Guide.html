Style Guide for Verified Teams

VISUAL DESIGN | DOCUMENTATION

DURATION

2 months

MY ROLE

Product Designer

PROJECT CONTEXT

Collaborated with Product Manager (PM), Senior Visual Designer, Lead Product Designer - Design Systems, Design Manager.

SUMMARY

In the expansive network of Postman, numerous API producers with similar identities posed a challenge for API consumers in distinguishing the authentic ones. I led the task of collaborating with the Product Manager (PM) and other designers to address this issue. Our goal was to design and articulate a style guide for a verified icon, affirming the authenticity, utility, and trustworthiness of API teams. I personally crafted the icon to align with Postman's overarching iconography and detailed a style guide, specifying the icon's specifications and usage guidelines. Additionally, I designed an email template for notifying API producers upon receiving the verified icon. As a result of these efforts, the discovery of authentic APIs saw a notable improvement of 5.8%.
